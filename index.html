Sobre as linguagens que ensino:

  • Python: é uma linguagem de programação de alto nível, interpretada e de propósito geral.
    É bastante popular devido à sua facilidade de uso, sintaxe simples e expressiva, além de possuir uma
    vasta biblioteca padrão e ser utilizada em áreas como ciência de dados, inteligência artificial e desenvolvimento web.

  • Java: é uma linguagem de programação interpretada, voltada principalmente para o desenvolvimento web.
    É utilizada para criar interatividade em páginas da web, animações, validações de formulários e muitas outras funcionalidades.

  • C#: é uma linguagem de programação orientada a objetos, desenvolvida pela Microsoft.
    É utilizada para desenvolvimento de aplicativos desktop, jogos e aplicações web. Possui recursos
    avançados, como gerenciamento automático de memória e suporte a tipos de dados abstratos.

  • R: é uma linguagem de programação de código aberto voltada para a análise estatística e visualização de dados,
    usado em diversas áreas, como biologia, genética, finanças, marketing, ciências sociais, entre outras, sendo uma ferramenta
    essencial para análise de dados em muitos setores. Por esses motivos, o R tem se tornado cada vez mais popular como uma ferramenta de análise de dados.
